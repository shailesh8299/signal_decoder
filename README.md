# Signal Decoder — Frontend Assignment

Small React + TypeScript game: users watch flashing patterns on a 5×5 grid and then try to select the squares that were flashing.

## Features
- 5×5 grid with multiple rule-based levels
- Show (flashing) → Selection → Check Answer → Feedback flow
- Light / Dark theme toggle (preference stored in localStorage)
- Clean, modular React + TypeScript code
